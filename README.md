# Work Done by Weight — Physics Slide Deck

An interactive, self-contained slide presentation that teaches students how to
calculate the **work done by weight (gravity)** as an object moves.

The deck walks through the physics step by step — the meaning of work, how the
direction of motion relative to the gravitational force affects the sign of the
work done, and fully worked example problems — in a clean, mobile-friendly
format suitable for use in class or for student self-study.
