## Q1 (6 pts) — Forces on a block
Given: m = 4 kg, F₁ = +18 N (right), F₂ = −6 N (left), frictionless.
- a) Fnet = F₁ + F₂ = (+18) + (−6) = **+12 N (to the right)**
- b) a = Fnet / m = 12 / 4 = **+3 m/s² (to the right)**
- c) **Unbalanced** — because Fnet ≠ 0 (Fnet = +12 N), the block accelerates right.

## Q2 (6 pts) — Suitcase in an elevator
Given: m = 25 kg, g = 10, elevator at rest.
- a) Fg = m·g = 25 × 10 = **250 N (downward)**
- b) At rest, a = 0 → Fnet = 0 → Fn − Fg = 0 → **Fn = 250 N (upward)**

## Q3 (5 pts) — Worker pulls a toolbox
Given: F_pull = 50 N (horizontal), f = 15 N (opposes), d = 20 m.
- a) Pull parallel to displacement → θ = 0°, cos0 = 1.
     W_pull = F_pull·d·cos0 = 50 × 20 × 1 = **+1000 J**
- b) Friction opposes → θ = 180°, cos180 = −1.
     W_f = f·d·cos180 = −(15)(20) = **−300 J**
- c) W_net = W_pull + W_f = (+1000) + (−300) = **+700 J**

## Q4 (6 pts) — Work–energy theorem
Given: m = 3 kg, vᵢ = 4 m/s, W_net = +30 J.
- a) KEᵢ = ½·m·vᵢ² = ½ × 3 × 4² = ½ × 3 × 16 = **24 J**
- b) W_net = KE_f − KEᵢ → KE_f = W_net + KEᵢ = 30 + 24 = **54 J**

## Q5 (6 pts) — Energy of a falling ball
Given: m = 0.5 kg, h = 5 m, g = 10, released from rest (vᵢ = 0), ground h = 0.
- a) PEg = m·g·h = 0.5 × 10 × 5 = **25 J**; at release vᵢ = 0 → **KE = 0 J**
- b) ME = KE + PEg = 0 + 25 = **25 J**
