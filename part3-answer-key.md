## Q1 (6 pts) — Forces on a block
Given: m = 4 kg, F₁ = +18 N (right), F₂ = −6 N (left), frictionless.
- a) Fnet = F₁ + F₂ = (+18) + (−6) = **+12 N (to the right)**
- b) a = Fnet / m = 12 / 4 = **+3 m/s² (to the right)**
- c) **Unbalanced** — because Fnet ≠ 0 (Fnet = +12 N), the block accelerates right.

## Q2 (6 pts) — Suitcase in an elevator
Given: m = 25 kg, g = 10, elevator at rest.
- a) Fg = m·g = 25 × 10 = **250 N (downward)**
- b) At rest, a = 0 → Fnet = 0 → Fn − Fg = 0 → **Fn = 250 N (upward)**

## Q3 (5 pts) — Worker pulls a toolbox
Given: F_pull = 50 N (horizontal), f = 15 N (opposes), d = 20 m.
- a) Pull parallel to displacement → θ = 0°, cos0 = 1.
     W_pull = F_pull·d·cos0 = 50 × 20 × 1 = **+1000 J**
- b) Friction opposes → θ = 180°, cos180 = −1.
     W_f = f·d·cos180 = −(15)(20) = **−300 J**
- c) W_net = W_pull + W_f = (+1000) + (−300) = **+700 J**

## Q4 (6 pts) — Work–energy theorem
Given: m = 3 kg, vᵢ = 4 m/s, W_net = +30 J.
- a) KEᵢ = ½·m·vᵢ² = ½ × 3 × 4² = ½ × 3 × 16 = **24 J**
- b) W_net = KE_f − KEᵢ → KE_f = W_net + KEᵢ = 30 + 24 = **54 J**

## Q5 (6 pts) — Energy of a falling ball
Given: m = 0.5 kg, h = 5 m, g = 10, released from rest (vᵢ = 0), ground h = 0.
- a) PEg = m·g·h = 0.5 × 10 × 5 = **25 J**; at release vᵢ = 0 → **KE = 0 J**
- b) ME = KE + PEg = 0 + 25 = **25 J**

## Q6 (5 pts) — Momentum of a ball
Given: m = 2 kg, v₁ = +8 m/s (right), v₂ = −8 m/s (left); right positive.
- a) p₁ = m·v₁ = 2 × (+8) = **+16 kg·m/s (to the right)**
- b) p₂ = m·v₂ = 2 × (−8) = **−16 kg·m/s (to the left)**
- c) Δp = p₂ − p₁ = (−16) − (+16) = **−32 kg·m/s (directed to the left)**

## Q7 (6 pts) — Impulse on a cart
Given: F = +40 N, m = 5 kg, vᵢ = 0 (at rest), Δt = 0.5 s, frictionless.
- a) J = F·Δt = 40 × 0.5 = **+20 N·s**
- b) pᵢ = m·vᵢ = 5 × 0 = **0 kg·m/s** (cart starts at rest)
- c) J = Δp = p_f − pᵢ → p_f = pᵢ + J = 0 + 20 = **+20 kg·m/s** (direction of applied force)

## Q8 (6 pts) — Baseball struck by a bat
Given: m = 0.15 kg, vᵢ = +30 m/s, v_f = −20 m/s, Δt = 0.02 s.
- a) pᵢ = m·vᵢ = 0.15 × (+30) = **+4.5 kg·m/s**;
     p_f = m·v_f = 0.15 × (−20) = **−3.0 kg·m/s**
- b) Δp = p_f − pᵢ = (−3.0) − (+4.5) = **−7.5 kg·m/s** (opposite to initial motion)
- c) F = Δp / Δt = (−7.5) / (0.02) = **−375 N** (opposite to the ball's initial direction)

## Q9 (7 pts) — Two skaters push off (conservation of momentum)
Given: m₁ = 60 kg (v₁ᵢ = 0), m₂ = 40 kg (v₂ᵢ = 0); after push v₁f = −2 m/s.
Left negative, right positive.
- a) Both at rest before → **p_total,i = 0 kg·m/s** (system initially at rest).
- b) 0 = m₁·v₁f + m₂·v₂f → 0 = (60)(−2) + (40)·v₂f → 0 = −120 + 40·v₂f
     → v₂f = +120 / 40 = **+3 m/s (to the right** — opposite to the 60 kg skater)

## Q10 (7 pts) — Car collision (perfectly inelastic)
Given: m_A = 1200 kg, v_Ai = +25 m/s (east); m_B = 1800 kg, v_Bi = 0; cars lock.
- a) p_total,i = m_A·v_Ai + m_B·v_Bi = (1200)(+25) + (1800)(0)
     = **+30,000 kg·m/s (east)**
- b) m_A·v_Ai + m_B·v_Bi = (m_A + m_B)·v_f → +30,000 = 3000·v_f
     → v_f = 30,000 / 3000 = **+10 m/s (east)**

## Bonus (3 pts — Extra Credit) — Braking with impulse
Given: m = 1500 kg, vᵢ = +20 m/s, v_f = 0, F = −300 N.
- J = Δp = m·v_f − m·vᵢ = (1500)(0) − (1500)(+20) = **−30,000 kg·m/s**
- J = F·Δt → Δt = J / F = (−30,000) / (−300) = **100 s**
